# Banco de Dados Epidemiológico

Coleta dados de dengue usando Python e salva em formato JSON para análise.

## Como funciona
- Execute o script `data-science/fetch_dengue.py` para baixar e processar os dados.
- O resultado será salvo em `backend/data.json`.

## Requisitos
- Python 3
- pandas (`pip install pandas`)

## Personalização
Edite o arquivo `fetch_dengue.py` para mudar o município ou período dos dados.
